# 🎚️ STREAM//RAW — AUDIO_DATABASE

A web-based audio streaming platform styled with a raw **neo-brutalist** aesthetic, focusing on high contrast, heavy typography, and bold layouts. This project was developed as an interactive portfolio piece to demonstrate complex state management and modern React architecture.

---

## ⚡ Key Features

* **Responsive Brutalist Grid:** Fluid layout for track cards featuring concrete click reactions and responsive text scaling via CSS `clamp`.
* **Dynamic Routing:** A modular sidebar integrated with `react-router-dom` to switch instantly between the main audio catalog and your custom **Favorites** list.
* **Global State Management (AudioContext):** Persistent audio playback tracking, a unified master pause/play controller, and track-saving capabilities utilizing React's Context API.
* **Genre Filter Array:** A perfectly centered, responsive filtering system for instant real-time track parsing (Reggae, Hip Hop, Rock Metal, Indie Pop).
* **100% Responsive Flow:** Mobile menu drawer with smooth Bezier-curve transitions and automated grid restructuring for smartphones.

---

## 🛠️ Tech Stack

* **React 18** (Functional components & Custom Hooks)
* **Vite** (Ultra-fast build tool and development server)
* **React Router DOM** (Production-ready single-page application routing)
* **Lucide React** (High-fidelity vector icons)
* **Pure CSS3** (Grid Layout, advanced Flexbox, and Mobile-First Media Queries)
